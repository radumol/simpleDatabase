
To set-up and run the app:

1)Have mongodb server running
2)Install necessary modules(should be already in the file):
  >npm install hat
  >npm install body-parser
  >npm install cookie-parser
3)Start the server with >node server.js
4)In the browser navigate to localhost:2406
5)Register new account
6)Start inputing recipes to the database
7)To log out, delete cookies and refresh the page

+developed on OS: Windows10
+tested on browser: Chrome
